      21             : // listener in several cases.
      22             : //
      23             : // The WaitableEvent maintains a list of waiters, protected by a lock. Each
      24             : // waiter is either an async wait, in which case we have a Task and the
      25             : // MessageLoop to run it on, or a blocking wait, in which case we have the
