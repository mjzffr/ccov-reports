     952             :  * http://dev.w3.org/csswg/css3-2d-transforms/#animation
     953             :  * This, in turn, refers to the unmatrix program in Graphics Gems,
     954             :  * available from http://tog.acm.org/resources/GraphicsGems/ , and in
     955             :  * particular as the file GraphicsGems/gemsii/unmatrix.c
     956             :  * in http://tog.acm.org/resources/GraphicsGems/AllGems.tar.gz
     957             :  *
     958             :  * The unmatrix reference is for general 3-D transform matrices (any of the
     959             :  * 16 components can have any value).
     960             :  *
     961             :  * For CSS 2-D transforms, we have a 2-D matrix with the bottom row constant:
     962             :  *
     963             :  * [ A C E ]
     964             :  * [ B D F ]
     965             :  * [ 0 0 1 ]
     966             :  *
     967             :  * For that case, I believe the algorithm in unmatrix reduces to:
     968             :  *
     969             :  *  (1) If A * D - B * C == 0, the matrix is singular.  Fail.
     970             :  *
     971             :  *  (2) Set translation components (Tx and Ty) to the translation parts of
     972             :  *      the matrix (E and F) and then ignore them for the rest of the time.
     973             :  *      (For us, E and F each actually consist of three constants:  a
     974             :  *      length, a multiplier for the width, and a multiplier for the
     975             :  *      height.  This actually requires its own decomposition, but I'll
     976             :  *      keep that separate.)
     977             :  *
     978             :  *  (3) Let the X scale (Sx) be sqrt(A^2 + B^2).  Then divide both A and B
     979             :  *      by it.
     980             :  *
     981             :  *  (4) Let the XY shear (K) be A * C + B * D.  From C, subtract A times
     982             :  *      the XY shear.  From D, subtract B times the XY shear.
     983             :  *
     984             :  *  (5) Let the Y scale (Sy) be sqrt(C^2 + D^2).  Divide C, D, and the XY
     985             :  *      shear (K) by it.
     986             :  *
     987             :  *  (6) At this point, A * D - B * C is either 1 or -1.  If it is -1,
     988             :  *      negate the XY shear (K), the X scale (Sx), and A, B, C, and D.
     989             :  *      (Alternatively, we could negate the XY shear (K) and the Y scale
     990             :  *      (Sy).)
     991             :  *
     992             :  *  (7) Let the rotation be R = atan2(B, A).
     993             :  *
     994             :  * Then the resulting decomposed transformation is:
     995             :  *
     996             :  *   translate(Tx, Ty) rotate(R) skewX(atan(K)) scale(Sx, Sy)

     999             :  * functions (i.e., all functions other than matrix()), in isolation,
